Setup Instructions:
1. Ensure python 3.7 is installed on the system (Does not work with any other minor version)
2. Ensure java 1.8 or later is installed and added to path (System uses mallet wrapper that has java as a dependency)
3. Inside python_deps folder run pyenv ./Script/activate (Set packages)
4. To scrape more articles can run scrape_articles.py. Currently scraping articles for the independent and the daily mail are supproted
